=== Event Handler ===

Event Handler is ability to user add new event to front side, it's available with custom php form and jQuery + php validation.

== Description ==

Step 1:
First install plugin after that you have show "Event" custom post type in your dashboard
After that change your Permalink Settings "Plain" to "Post name" 

Step 2:
Use below shortcode for display event custom form on frontend page
[event_custom_form]

Step 3:
Use below shortcode for display event list by event city. on frontend page
[get_event_list]
