=== Event Handler ===

Event Handler is ability to user add new event to front side, it's available with custom php form and jQuery validation.

== Description ==

Step 1:
First install plugin after that you have show "Event" custom post type in your dashboard

Step 2:
Create "Event Form" Page in dashboard and note that name must be same as describe.

Step 3:
Add below shortcode for display event by event city wise.
[event-city city_name="cityname"][/event-city]
